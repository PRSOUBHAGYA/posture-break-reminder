# PostureGuard

A macOS menu bar application that monitors sitting posture in real time using MediaPipe and OpenCV.

## Setup

1. **Create virtual environment:**
   ```bash
   python3 -m venv venv
   source venv/bin/activate
   ```

2. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the app:**
   ```bash
   python main.py
   ```

## Permissions
The app requires Camera and Accessibility permissions to function correctly.
